## 🎯 Propósito de Aplicación
Integrar en un proyecto real todos los conocimientos para construir una landing page completa, manipular el DOM con JavaScript, validar formularios y publicar en producción usando GitHub Pages u otro hosting gratuito, documentando el proceso con commits.

---

## 🛠️ Tecnologías Utilizadas
* **HTML5:** Estructura semántica de la página.
* **CSS3:** Estilos, variables (Custom Properties) y diseño responsivo adaptado a múltiples dispositivos.
* **JavaScript (Vanilla):** Lógica del lado del cliente, manipulación del DOM y validación de formularios en tiempo real.
* **Git & GitHub:** Control de versiones y alojamiento para despliegue en producción.

---

## ✨ Características Principales
* **Diseño Responsivo:** Interfaz adaptable a dispositivos móviles, tablets y pantallas de escritorio.
* **Menú Interactivo:** Funcionalidad de menú hamburguesa para una navegación fluida en dispositivos móviles.
* **Validación de Formulario Dinámica:** Comprobación de campos (nombre, email, teléfono, fechas) mediante JavaScript antes de la simulación de envío, brindando retroalimentación visual instantánea al usuario (mensajes de error y acierto).
* **Manipulación del DOM:** Actualización de elementos de la interfaz en tiempo real, como la visualización del mensaje de éxito al cotizar y el cálculo automático del año en el pie de página.
